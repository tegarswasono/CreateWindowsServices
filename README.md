## Cara Install
<ol>
<li>Buka CMD dan masuk folder .net Framework mu (C:\Windows\Microsoft.NET\Framework\v4.0.30319)</li>
<li>Install Service (InstallUtil.exe + Your path + \your service name + .exe)</li>
<li>Done</li>
</ol>
